This is a fork of the checker framework repository for developement of an IndexChecker that checks for legal array/list accesses.

If you wish to use the Checker Framework in general, see file README.txt.

For using or developing the IndexChecker, find our manuals located in the directory:
	checker-framework/checker/manual-IndexChecker/
They can also be found online hosted on github at:
	https://github.com/gyhughes/checker-framework/tree/master/checker/manual-IndexChecker/

Our developer's website can be found at:
	http://students.washington.edu/gyhughes/cse403/TeamWebsite/
